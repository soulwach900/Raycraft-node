# Raycraft-node

## Description

Simple Template of Minecraft using Raylib and Nodejs

## Installation From Source

### You Need NODEJS

1. **Clone the repository:**

   ```bash
   git clone https://github.com/soulwach900/Raycraft-node.git
   ```

   ```bash
   cd "Raycraft-node"
   ```

2. **Running:**

   ```bash
   npm run start
   ```
